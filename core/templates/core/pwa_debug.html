📋 Como Validar o PWA

Chrome/Edge (Desktop):

  1. Abra as DevTools (F12)
  2. Vá para a aba "Application"
  3. Verifique "Manifest" - todos os campos devem estar corretos
  4. Verifique "Service Workers" - deve estar ativo
  5. Procure o ícone de instalação na barra de endereços (canto direito)

Android (Chrome):

  1. Acesse o app via Chrome
  2. Verifique se o banner de instalação aparece
  3. Teste se o app pode ser adicionado à tela inicial
  4. Abra as DevTools (menu → Mais ferramentas → Ferramentas do desenvolvedor)
  5. Verifique o manifest e service worker na aba "Application"

Observações para Produção:

  • O app deve ser servido via HTTPS (exceto localhost)
  • Os ícones devem estar disponíveis nos caminhos especificados no manifest
  • O service worker deve estar registrado e ativo
  • O manifest deve ser válido e acessível
